# Parad0x Compress
### On-device image & video compression (no cloud uploads, no accounts)

## What it is
- Compress photos and videos directly on your phone.
- Privacy-first: your media never leaves the device.

## Features
- **Free**: single file only • Video ≤200MB • Photo ≤10MB
- **Pro**: batch mode + higher limits + **AVIF** output

## Supported formats
**Input**
- Images: JPG, PNG, WebP
- Videos: MP4, MOV

**Output**
- **Pro**: AVIF, WebP, JPG, MP4
- **Free**: WebP, JPG, MP4

## Pro tips
- **AVIF** = best quality/size, but slower to encode.
- For big batches, set battery to **Unrestricted**.
